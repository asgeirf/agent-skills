# Agent Skills

A collection of skills for Claude Code and other AI coding agents.

## Skills

| Skill | Description |
|-------|-------------|
| [flow-graph](skills/flow-graph/) | Generate animated GIF/WebM videos of graph visualizations |

## Installation

```bash
npx skills add asgeirf/agent-skills
```

To install a specific skill:

```bash
npx skills add asgeirf/agent-skills@flow-graph
```

## Prerequisites

- Node.js 20+
- ffmpeg (`brew install ffmpeg`)
